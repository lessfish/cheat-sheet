

| key                                | 作用                               |
| ---------------------------------- | ---------------------------------- |
| cmd+d                              | 水平方向扩展窗口                   |
| cmd+shift+d                        | 竖直方向扩展窗口                   |
| cmd+点击文件                       | 会用默认程序打开文件               |
| cmd+t                              | 新建 tab                           |
| cmd+← / cmd+→                      | 切换 tab                           |
| cmd+[ / cmd+]. 或者 cmd+option+←/→ | 切换 pane                          |
| cmd+option+e                       | 全屏所有 tab                       |
| cmd+/                              | 高亮当前的鼠标位置                 |
| cmd+r                              | 清屏                               |
| cmd+;                              | 查看历史命令                       |
| cmd+shit+h                         | 查看剪切板历史                     |
| command+enter                      | 全屏切换                           |
| control+u     | 删除光标前的内容          |
| control+a     | 光标到行首             |
| control+e     | 光标到行尾             |
| control+w     | 删除光标之前的单词         |
| 双击/三击                          | 双击选中，三击选中整行。选中即复制 |
